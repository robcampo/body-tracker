# Body Tracker

A mobile web app for tracking body measurements over time.

🌐 **Live app:** https://robcampo.github.io/body-tracker/

## Features

- **Dashboard** — overview of all tracked measurements with trend charts and encouraging messages
- **Log entries** — record measurements for any date
- **History** — browse and delete past entries
- **Charts** — detailed chart for any single measurement with date range filtering
- **Trend detection** — automatically detects if you're losing, gaining, or maintaining

## Measurements

Weight, waist, hips, chest, arms, thighs, neck, shoulders, and calves.

## Add to your home screen

**iPhone/iPad:** open the link in Safari → Share → Add to Home Screen

**Android:** open the link in Chrome → menu → Add to Home Screen

## Tech

- [Vue 3](https://vuejs.org/) — UI framework
- [Chart.js](https://www.chartjs.org/) — charts
- localStorage — all data stays on your device, nothing is sent anywhere
